# My Portfolio

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Skills section with categorized technologies
- Projects showcase with live demo and code links
- Contact information section

## Deployment

This portfolio can be easily deployed to GitHub Pages for free hosting.

### Steps to Deploy:

1. **Create a GitHub Repository:**
   - Go to GitHub and create a new repository
   - Name it something like `yourusername.github.io` for a custom domain, or any name for `username.github.io/repository-name`

2. **Upload Your Files:**
   - Clone the repository to your local machine
   - Copy the `index.html`, `style.css`, and `work.js` files to the repository
   - Commit and push the changes

3. **Enable GitHub Pages:**
   - Go to your repository settings
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click "Save"

4. **Access Your Site:**
   - GitHub will provide a URL like `https://yourusername.github.io`
   - Your portfolio will be live at that URL

## Customization

- Update the content in `index.html` with your personal information
- Modify `style.css` to change colors, fonts, or layout
- Add your actual projects to the `projects` array in `work.js`
- Replace the placeholder image in the About section with your photo
